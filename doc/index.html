FermiSurfer

FermiSurfer is an visualization tool for Fermi surfaces. This application displays Fermi surfaces colored as a function of an arbitrary scalar quantities such as magnitude of Fermi velocities and superconducting gap.

News

  • 2017/8/16 FermiSurfer 1.9.1 is released !

    In this update, bug-fix and speed-up are performed. For more details, please see the release note.

  • 2017/3/19 Interface for QuantumESPRESSO-6.1 is released !

    We release a patch for QE-6.1 to add routines for displaying the Fermi velocity and the orbital character on Fermi surfaces. This can be obtain in the release page in QEForge. This is a part of the DFPT-tetra project.
