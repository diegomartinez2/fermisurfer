FermiSurfer

FermiSurfer is an visualization tool for Fermi surfaces. This application displays Fermi surfaces colored as a function of an arbitrary scalar quantities such as magnitude of Fermi velocities and superconducting gap.

For citing FermiSurfer

We would appreciate if you cite the following article in your research with FermiSurfer.

"FermiSurfer: Fermi-surface viewer providing multiple representation schemes", M. Kawamura, Comp. Phys. Commun 239, 197 (2019) (arXiv:1811.06177).
